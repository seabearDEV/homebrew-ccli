# Homebrew Tap for CodexCLI

This is the official [Homebrew](https://brew.sh/) tap for [CodexCLI](https://github.com/seabearDEV/codexCLI).

## Installation

```bash
brew tap seabeardev/ccli
brew install ccli
```

## Updating

```bash
brew upgrade ccli
```

## About

CodexCLI is a command-line information store for quick reference of frequently used data. It uses hierarchical dot notation for organizing structured data, with features like aliases, interpolation, encryption, and shell tab-completion.

For more information, see the [main repository](https://github.com/seabearDEV/codexCLI).
